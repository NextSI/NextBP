# Inclusão de ação "Copiar para um novo usuário" na Listagem de Usuários
16/10/2025
## Melhoria
## Referência: BPM40671
## Módulo: Cadastros Básicos
***

Foi adicionada a opção **"Copiar para um novo usuário"** no menu de contexto da listagem de usuários.  
A funcionalidade permite duplicar um usuário existente, criando um novo registro com base nas informações do usuário original, exceto pelo campo **username**, que deverá ser informado manualmente.  

Ao salvar o novo usuário, todas as **permissões** do usuário de origem são automaticamente copiadas, agilizando o processo de criação e configuração de novos usuários com perfis semelhantes.
