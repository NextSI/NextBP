# Configuração para permitir salvar credencias do usuário no navegador
13/02/2025
## Melhoria
## Referência: CHM74395
## Módulo: Geral
***

Adicionado em `System.config.php` a possibilidade de utilizar o parâmetro `LOGIN_EXIBIR_OPCAO_SALVAR_USUARIO` para definir se a tela de login exibe as opções de salvar o usuário no navegador. Por padrão este parâmetro é `true`.

No exemplo abaixo está sendo desativada a opção de salvar as credências do usuário no navegador:
```php
define('LOGIN_EXIBIR_OPCAO_SALVAR_USUARIO', false);
```
