# Atualização API Clicksign
13/10/2025
## Melhoria
## Referência: BPM30828
## Módulo: Configurações
***

A integração com a Clicksign foi atualizada para refletir as mudanças mais recentes da plataforma.
Essa atualização garante compatibilidade com a versão atual da API e melhora a estabilidade do processo de assinatura digital.

Principais ajustes realizados:

- Adequação ao novo padrão de endpoints da API Clicksign.
- Atualização dos formatos de requisição e autenticação.
- Correções no envio de envelopes, documentos e notificações.
- Tratamento de erros conforme o novo padrão de respostas da API.
