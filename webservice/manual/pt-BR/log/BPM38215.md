# Filtro de Quadros no Módulo de Tarefas  
08/10/2025  
## Melhoria
## Referência: BPM38215
## Módulo: Tarefas
***

Foi adicionada uma nova funcionalidade para filtrar os quadros dentro do módulo de Tarefas. Agora, o usuário pode selecionar entre as seguintes opções de visualização:

- **Meus Quadros**: Exibe apenas os quadros nos quais o usuário logado é membro da equipe ou é seu quadro particular.
- **Quadros que Criei**: Exibe todos os quadros criados pelo usuário logado desde que ele faça parte da equipe ou seja um quadro particular.
- **Todos os Quadros**: Lista todos os quadros cadastrados no sistema. Essa opção está disponível apenas para usuários com perfil de Administrador.
