# Ajuste ao gerar PDF de solicitação de processos
11/03/2025
## Correção
## Referência: BPM34960
## Módulo: Processos
***

Mensagem de erro: `TCPDF ERROR: Please explicitly set action attribute path!`.

Ao copiar conteúdo de uma página de um navegador qualquer e colar em um editor de texto do Next BP, a estrutura HTML é copiada e junto com ela podem vir elementos inválidos para gerar o PDF. 

Adicionado processamento para ignorar o elemento `<form>` ao gerar o PDF da solicitação.
