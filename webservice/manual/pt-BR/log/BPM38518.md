# Adicionada rotina para exclusão de parâmetros de usuários
13/05/2025
## Melhoria
## Referência: BPM38518
## Módulo: Configurações
***

Adicionada rotina para manutenção e exclusão de parâmetros de usuários no Next BP. Acessível pelo menu "Parâmetros de Usuários".

1) A visualização e exclusão dos parâmetros de todos os usuários está disponível apenas para usuários do tipo Administrador.
2) Para usuários que não são administradores, a funcionalidade pode ser habilitada nas permissões de usuário, em "Configurações", "Usuários", "Editar permissões do usuário". Dessa forma, será possível realizar a manutenção dos próprios parâmetros.
