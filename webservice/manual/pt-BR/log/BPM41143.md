# Ajuste na notificação por e-mail aos signatários de documentos via Clicksign
27/10/2025
## Correção
## Referência: BPM41143
## Módulo: Documentos
***

Recurso para notificar signatários da Clicksign possuia falha de comunicação com a API da Clicksign.

Com o ajuste na comunicação com a API da Clicksign, as notificações para signatários voltaram a ser enviados.
