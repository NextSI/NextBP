# Ponto de Entrada ao Salvar Cartão.
14/07/2025
## Melhoria
## Referência: BPM38221
## Módulo: Tarefas
***

Foram incluídos dois novos Pontos de Entrada no processo de salvamento de cartões. Ambos são acionados no momento em que o cartão é salvo, permitindo a personalização em dois estágios distintos:

1. **Antes de salvar** (antes de efetivar as alterações);
2. **Depois de salvar** (após efetivar as alterações).

Com essa melhoria, é possível realizar validações e aplicar lógicas personalizadas antes e depois do salvamento do cartão.

Exemplo:

```
PE::add('tarefa_quadro_lista_cartao_salvar_antes', function($tarefa_quadro_lista_cartao_model) {
    // seu código aqui
});

PE::add('tarefa_quadro_lista_cartao_salvar_depois', function($tarefa_quadro_lista_cartao_model) {
    // seu código aqui
});
```
