# Integração com RD Station MKT Webhook
14/05/2025
## Melhoria
## Referência: BPM36429
## Módulo: CRM
***

Adicionada integração via webhook para integração de leads da plataforma RD Station.

Configure o webhook através do menu "Configurações", "Integrações", "RD Station Webhook Mkt".

Uma vez configurado, quando o RD Station enviar uma comunicação para a URL Webhook, o Next BP irá:

1) Criar Contato;
2) Criar Organização;
3) Relacionar Contato e Organização;
4) Criar Oportunidade;
5) Salvar nas observações da oportunidade os dados de campos customizados do RD Station.

Requisito para funcionamento: habilitar o Job Scheduler `Fila_Evento.job.php`.
