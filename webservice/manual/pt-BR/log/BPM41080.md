# Criar rotina para copiar os grupos de usuário de um usuario para outro usuario
05/11/2025
## Melhoria
## Referência: BPM41080
## Módulo: Configurações
***

Criada nova rotina que permite **copiar os grupos de um usuário para outro**.  
Essa funcionalidade tem como objetivo facilitar a configuração de permissões e perfis de acesso, reduzindo o tempo necessário para replicar grupos entre usuários com funções semelhantes.

Para acessa-la, na listagem de usuários utilize a opção "Copiar de outro usuário...".
