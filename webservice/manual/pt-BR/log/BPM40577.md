# Ajuste em obrigatoriedade do campo Domínio no cadastro de Servidores LDAP
04/09/2025
## Correção
## Referência: BPM40577
## Módulo: Geral
***

No cadastro de "Servidores LDAP", foi adicionado a obrigatoriedade de informar o campo "Domínio" quando campo "Tipo" for "Microsoft Active Directory (AD)".

Para facilitar a experiência de quem está configurando o servidor LDAP, foi adicionado o botão "Testar Filtro". Dessa forma o botão "Testar Conexão" irá testar somente a conexão com o servidor LDAP, e o botão "Testar Filtro" executará o filtro no servidor LDAP, retornando a quantidade de entradas resultadas da consulta.
