# Adicionado ponto de entrada ao salvar um projeto
10/03/2025
## Melhoria
## Referência: CHM74804
## Módulo: Projetos
***

Adicionados pontos de entrada PHP que poderão ser executados antes e depois de salvar um projeto.

```php
PE::add('projeto_salvar_antes', function(&$projeto_model) {
    // seu código
});
```

```php
PE::add('projeto_salvar_depois', function($projeto_model) {
    // seu código
});
```

As funções `Sys\Util::isGerente` e `Sys\Util::isExecutor` foram migradas para `Projeto_DAO::isGerente` e `Atividade_Projeto_DAO::isExecutor`.
