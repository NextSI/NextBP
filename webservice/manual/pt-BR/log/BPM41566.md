# Job de importação automática de usuários do LDAP
11/03/2026
## Melhoria
## Referência: BPM41566
## Módulo: Configurações
***

**O que foi melhorado:**

Nesta versão foi criado o novo job `Importacao_Usuarios_LDAP_Automatica.job.php` para realizar a importação automática de usuários sem depender do job manual já existente.

- Quando o usuário retornado pelo LDAP já possui **e-mail existente** no BP, o registro **não é atualizado** e também não é recriado.
- Quando o usuário retornado pelo LDAP possui **e-mail ainda não cadastrado** no BP, um novo usuário do tipo **Colaborador** é criado automaticamente.
- Os novos usuários criados por esse job já são vinculados ao servidor LDAP de origem.
- Durante a criação são aplicadas as configurações padrão de **Área**, **Nível**, **Calendário de Trabalho**, **Empresa padrão do sistema** e **Permissões padrão de colaboradores**.
- Usuários LDAP sem e-mail, com login LDAP já vinculado ou com conflito de `username` no BP são ignorados e registrados no log do job.
- Apenas os servidores LDAP que possuírem a configuração de importação automática habilitada serão processados.

**Agendamento sugerido:**

Para execução automática a cada 1 hora no Job Scheduler, configurar o script `Importacao_Usuarios_LDAP_Automatica.job.php` com:

- `Minuto`: `0`
- `Hora`: `*/1`
- `Dia`: `*`
- `Mês`: `*`
- `Dia da semana`: `*`
