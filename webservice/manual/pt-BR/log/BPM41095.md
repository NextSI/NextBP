# Ajuste na análise do fluxo do gateway com campos decimais
30/10/2025
## Correção
## Referência: BPM41095
## Módulo: Processo
***

Na modelagem do gateway, ao definir uma regra de automação considerando campos decimais com valores superiores a sete casas, o fluxo das solicitações não avançava corretamente.

Com este ajuste a análise da automação de gateways passa a tratar corretamente os valores de campos decimais.

Para campos do tipo inteiro este tipo de problema não ocorria.
