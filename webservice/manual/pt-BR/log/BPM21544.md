Tela de Kanban para Visualização e Consulta de Atividades Analíticas do Projeto

# Filtros Disponíveis

1. Descrição da Atividade: Permite pesquisar cards utilizando partes do nome da atividade do projeto.
2. Responsável: Permite filtrar cards pelo executor ou responsável pela atividade.
3. Status: Permite filtrar cards com base no status das atividades.

Os filtros podem ser combinados para uma consulta mais precisa.

# Informações nos Cards
Cada card exibe as seguintes informações:

1. Nome da Atividade;
2. Data de Início;
3. Prazo para Conclusão;
4. Percentual de Conclusão;
5. Horas Trabalhadas;
6. Saldo de Horas;
7. Nome e Imagem dos Executores.

# Detalhamento das Atividades

Ao clicar em qualquer card, as informações completas da atividade serão exibidas no modo de visualização.
